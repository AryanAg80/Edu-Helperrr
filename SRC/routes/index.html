About

In today's world, which is full of distractions, students find it hard to concentrate. So, we have come up with a cool way of studying with your study assistant discord bot EDU-HELPER. To keep it more fun and engaging we have added some cool features.
Whenever the student completes his tasks, he is awarded edu-coins. It's a special type of token currency that is given as a reward for being consistent. Moreover, we have come up with a streak maintenance system so that you have the motivation to study daily and not break the streak!ðŸ”¥





Tech Specs:

  • Daily-Routine
  • To-do Reminder
  • Time-Constraint Sessions
  • Timers
  • Study Streaks
  • Terminal




  • Daily-Routine

    • Our Bot can schedule your daily tasks accordingly (Time provided by the user to remind).
    • It can remind you of all the tasks daily.
    • The more you do your tasks, the more you earn edu-coins.
    • Make a streak and earn 2x coins
    • You can also schedule your daily habits. Works as a habit reminder.




  • To-do Reminder

    • This feature of our bot will take input from the user about different tasks and
      assignments along with their corresponding deadlines.
    • Make a fancy task list.
    • Earn Edu-coins on every task done.
    • Reminds about the assignments and tasks a day before the deadline.





  • Time-Constraint Sessions

    • Make a private channel.
    • Select the number of times you want your session to go on.
    • Break and Study Time to get reminded.
    • Upload your Session Tasks Accordingly.
    • Earn upto 1.4x Edu-coins on every successful session.





  • Timers

    • Set a timer for a short interval of time.
    • Get personalized Direct message Reminder.
    • Get your Timer list as well.





  • Study Streaks

    • Join the voice channels and count your study time accordingly.
    • Earn Edu-coins on every minute studied.
    • Maintain a streak of days you studied in the channels.
    • Get reminded when you are inactive for a longer duration of time.
    • Get Personalized study-time board.




  • Terminal

    • Run code for multiple languages: BASH/SH, PHP, Python, Node.js, Bat, C, Java and more.
    • It uses a npm package which debugs the code and shows the output.
